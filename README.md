# Kenilworth Uyogo Friendship Link archives 🇬🇧🇹🇿

This repo contains scanned photos, documents and letters from the archives of [Kenilworth Uyogo Friendship Link](https://www.kenilworthuyogofriendshiplink.org/) (KUFL).

KUFL was founded in 1979 to promote mutual benefits between Kenilworth (Warwickshire, United Kingdom) and Uyogo (Tabora, Tanzania).
